# نبذة عن المشروع

تُرجمان يهدف إلى تحسين الترجمة والمشاركة فيها بطريقة تفاعلية مع المستخدمين عبر تسهيل الترجمة لتكون متوفرة على منصاتهم المفضلة، وجعلها اسرع واكثر متعة.

هذا المشروع تابع لمجتمع أسس https://aosus.org
